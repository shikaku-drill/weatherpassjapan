天気で決める、日本の遊び方

WEATHERPASS JAPANは、いまの空を見てその日の予定を組み立てるサイトです。東京・京都・大阪・北海道・沖縄など15都道府県の体験と名所147件を、現在地からどれだけ早く行けるか順に並べます。雨なら完全屋内が、猛暑なら冷房のある場所が、晴れなら屋外の名所が先頭に来ます。

雨の日に大事なのは一つだけ。そこへ行くまでに濡れるかどうかです。teamLab Planets、すみだ水族館、寿司握り体験、屋根付き商店街——天気に閉められない場所だけを表示します。カードには移動時間・乗換回数・駅からの徒歩が出るので、傘を差す時間が最も短い選択肢を選べます。35°Cを超える日も、同じ考え方で暑さを避けます。

料金はKlook・KKday・Viatorの3社へ直接リンクします。どこが安いかは施設と日によって変わるので、見比べてから予約してください。窓口で当日買える施設にはその旨を表示します。交通パスは、あなたの旅程に本当に必要なときだけ、パス無しの運賃概算と並べて出します。

対応エリア：東京 / 神奈川 / 千葉 / 京都 / 大阪 / 奈良 / 愛知 / 石川 / 広島 / 福岡 / 沖縄 / 北海道 / 宮城 / 青森 / 山形

雨の日・猛暑・予約のギモンに即答

東京で雨の日は何ができる？

実はたくさんあります。teamLab Planets、すみだ水族館、すみだ北斎美術館、寿司握り体験、屋根付き商店街など、完全屋内の体験は雨でも通常営業。WEATHERPASSはライブ予報を読み、今日まだ入場でき、今から間に合うものだけを表示し、Klook・KKday・Viatorの3社を並べて比較できます。

Klook・KKday・Viator、日本の観光はどこが最安？

施設と日によって変わります。各社が独自のセールを行うため、teamLabで安いサイトがスカイツリーでも安いとは限りません。だからWEATHERPASSは全施設から3社へ直接リンクします。最初に開いたサイトで決めず、3社を見比べてから予約してください。

日本の猛暑日はどう過ごすべき？

33〜38°Cの日は雨と同じ発想で「完全屋内・冷房あり」へ。デジタルアート美術館、水族館、屋内料理教室、スパ施設が定番です。WEATHERPASSは30°C超で自動的に猛暑モードに切り替わり、涼しい屋内体験を日差しから最速で逃げ込める順に表示します。
