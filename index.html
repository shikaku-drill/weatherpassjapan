使用條款

1. 關於本服務

WEATHERPASS JAPAN（以下稱「本服務」）是依據天氣資訊推薦日本國內體驗與活動，並提供 Klook、KKday、Viator 等外部訂票網站（以下稱「合作網站」）連結的資訊服務。本服務不進行任何預訂或付款；預訂契約直接成立於使用者與合作網站或設施之間。

2. 資訊正確性

營業時間、費用、所需時間與天氣預報僅供參考，本服務不保證其正確性、完整性或即時性。所顯示價格為概估，實際以合作網站顯示的價格為準。交通時間與旺季價格調整為推估值。訂票前請務必確認合作網站與設施官方資訊。

3. 禁止事項

  1. 妨礙本服務營運之行為
  2. 未經授權之存取或大量擷取（爬蟲）
  3. 未經授權重製、轉載或商業利用本服務內容
  4. 違反法令或公序良俗之行為

4. 智慧財產權

本服務所含設計、文字、標誌等智慧財產權，歸營運者或正當權利人所有。設施名稱與服務名稱為各權利人之商標。

5. 免責聲明

除營運者故意或重大過失外，對於因使用或無法使用本服務、於合作網站的預訂・付款・取消、設施營業狀況或天氣變化所生之損害，營運者不負賠償責任。若強制性消費者保護法規限制本免責條款，則於該法規允許範圍內適用。

6. 服務之變更與終止

營運者得不經事前通知，變更、中止或終止本服務。

7. 條款之變更

本條款得視需要修訂。修訂後之條款於刊登於本頁時生效。

本條款以日本法為準據法。關於本服務之爭議，以東京地方法院為第一審專屬合意管轄法院。

隱私權政策

1. 我們處理的資訊

位置資訊: 僅在您選擇「使用目前位置」時，才會透過瀏覽器的定位 API 取得經緯度。位置資訊在您的瀏覽器內用於距離計算與天氣查詢，不會儲存於營運者的伺服器。為查詢天氣，經緯度會傳送至氣象資料提供者（Open-Meteo）。

帳號與個人資料: 本服務不需註冊帳號，也不蒐集姓名、電子郵件等個人資料。

裝置內儲存（localStorage）: 為方便使用，「我的行程」與收藏會自動儲存在您瀏覽器的 localStorage（僅限您的裝置）。這些資料不會離開您的裝置，營運者與任何第三方均無法讀取。您可隨時透過「我的行程」分頁的「刪除已儲存資料」按鈕，或瀏覽器的清除瀏覽資料功能刪除。本服務本身不使用 Cookie。

2. 第三方服務

本服務使用以下第三方服務。各服務對資訊的處理依其自身隱私權政策辦理。

  1. Open-Meteo（氣象資料）— 會接收您的座標
  2. Google Fonts（字型配送）— 您的 IP 位址可能傳送至 Google
  3. Klook、KKday、Viator（預訂）— 離開本站後依各公司政策辦理

※若日後導入主機服務商、流量分析或廣告，可能記錄存取紀錄（IP 位址、瀏覽時間等）。屆時將更新本政策。

3. 您的權利

您可隨時於瀏覽器或裝置設定中拒絕或撤銷定位權限。如需依所在地法令（日本個資法、GDPR 等）行使權利，請透過下方聯絡方式與我們聯繫。

聯盟行銷揭露

本服務部分連結為聯盟行銷連結。若您經由這些連結於合作網站完成預訂，營運者可能自該合作網站取得推薦佣金，且不會增加您的支付金額。每項體驗均以相同條件連結至三家網站，佣金不影響排序，也不影響顯示哪些網站。哪一家最便宜會因設施與日期而異，因此本服務不做最低價判定，請您自行比較。
